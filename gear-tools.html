Tools I actually use

Short list. No padding.

I’m not interested in huge gear lists. These are the tools I actually use to run the channels. If something isn’t here, it means I don’t use it.

Affiliate disclosure: some links may be affiliate links. Updated: 2025

Editing

DaVinci Resolve

Free, powerful, and stable enough for long-form editing. I used it before I paid for anything else.

Descript (selective)

I only use it for rough cuts and transcription, not full edits. It’s useful when time is tight.

Analytics & workflow

YouTube Studio

The only analytics platform I trust because it’s the source. Everything else is a wrapper.

Notion

I keep video ideas, experiments, and retention notes in a single database. Simple and searchable.

Google Sheets

Still my favorite for tracking CTR tests and thumbnail variations over time.

Gear

I don’t list specific models because gear changes. If you want to spend money, spend it on audio first. Then lighting. The camera is third. Every time.

If I recommend a tool, it’s because I use it. If I ever stop using it, I’ll remove it. That’s the deal.
